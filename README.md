# TielOS

TielOS is a personal website made by me, orivio, using HTML, CSS, and JavaScript
It's a part of Stardance's WebOS 1 quest.

## Contents

### Apps
TielOS, as of now, only has three apps. These are:
- about
- blog
- peep

The about app is a standard about-me page. It contains snippets of info about things I enjoy. However, I tend to be fairly bad at concretely describing my hobbies (but don't worry, that gets covered).

The blog app is, again, a blog where I plan on uploading general life updates. It'll kind of be like a public journal, I guess! I'll keep it updated (at least I hope) over time, so that I can show others how far I've come in my endeavors!

PEEP. PEEP is literally just a tab where I infodump about some of my favorite species of birds. I talk about cockatoos and conures, and I plan to add more and more as time allows.

### Features
TielOS has a clock that uses JavaScript to access the user's calendar and display the date and time.
Windows can also be opened/closed with smooth animation via labelled buttons. They can also be smoothly dragged thanks to the glorious thing that is JavaScript
